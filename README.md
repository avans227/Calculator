# Calculator

A simple command-line calculator built with Python.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Modulus
- Power
- Square Root
- Handles invalid input
- Handles division by zero

## Example

text Select an operation: 1. Add 2. Subtract 3. Multiply 4. Divide 5.Modulus 6.Power 7.Square root
Enter choice: 1 
Enter first number: 10
Enter second number: 5 
Result: 15
